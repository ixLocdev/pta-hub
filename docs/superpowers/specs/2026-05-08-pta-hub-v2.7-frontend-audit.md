# PTA Hub v2.7.0 — Frontend Design Audit

**Date:** 2026-05-08
**Author:** Claude (audit), Lucas Deichl (approval)
**Status:** Approved for implementation
**Audience reminder:** PTA volunteer parents (often 35–65, often on phones) and content editors who use the admin maybe twice a month. Every fix should make something less intimidating, not more visually impressive.

## Cross-cutting findings

| ID | Issue | Tier |
|---|---|---|
| X1 | Four different "blues" across surfaces — `#2271b1` (Wizard + admin headers), `#2563eb` (front-end Best Answer button), `#3b82f6` (search focus + links), `#4f46e5` indigo (Print/Copy hover in `single.css:224`). Pick one primary and one accent — propose `#2563eb` as primary, `#1e40af` as deep accent. Replace globally via CSS variables. | T2 |
| X2 | Body text below 16px on critical surfaces — `.ptk-card-excerpt` 13px (`search-page.css:378`), `.ptk-card-desc` 12px (`content-wizard.css:127`), `.ptk-related-card-excerpt` 12px (`single.css:171`). Bump all to 14px minimum. | T2 |
| X3 | Hover/transition timings vary (0.1s / 0.15s / 0.2s). Standardize on 150ms ease. | T3 |
| X4 | Tag border-radius inconsistent — 8px on search cards, 12px on single-entry tags. Pick 999px (pill) globally. | T3 |

## Per-surface findings

### 1. Content Wizard
- **T1** Placeholder text doing the work of helper text — converts to visible `<p class="ptk-field-help">` below each label.
- **T1** No "Step 2 of 5" progress indicator.
- **T2** No legend for required `*`.
- **T2** Submit button looks like a generic WP "Save."
- **T3** No "you have unsaved changes" warning when leaving the page mid-form.

### 2. Post editor sidebar meta boxes
- **T1** Five separate PTK meta boxes in the sidebar. Group visually + collapse less-used ones by default.
- **T2** Inconsistent padding/borders across the five boxes — extract `.ptk-sidebar-block` class.
- **T2** Visibility meta box invisible when no restrictions set — always render with "Visible to everyone" default + expandable controls.

### 3. Network Sync admin page
- **T2** All inline styles — extract to `assets/css/network-sync.css`.
- **T2** "Sync all entries now" gives no progress feedback — add a confirm modal + status counter.
- **T3** "School Sites in Network" table doesn't show coverage — add a "Synced entries" column.

### 4. Suggestions queue
- **T2** "Convert to Draft" and "Discard" row actions look identical — style differently.
- **T3** No bulk actions.
- **T3** No friendly empty state.

### 5. Search Analytics + dashboard widgets
- **T2** Numbers without context — add inline CSS bars.
- **T2** "Needs Review" widget shows dates but no urgency dots — reuse the green/amber/red dots from the list table.
- **T3** 30-day window hardcoded — add a time-range selector.

### 6. Starter Content Importer
- **T2** "Already imported" warning has same weight as help text — promote to top yellow strip.
- **T3** No preview of imported entry titles — add accordion.

### 7. Search page + single entry + glossary (front-end)
- **T1** "PTA Hub" title with no explanation for first-time visitors — tighten subtitle.
- **T1** Fresh-install empty state — replace search box with "Coming soon" panel when zero entries.
- **T2** TOC component visually heavy — switch to left-border-only treatment.
- **T2** Print/FAQ-copy action bar below the title — move to top-right of title row on desktop.
- **T2** "Was this helpful?" after Related Entries — move above.
- **T2** Related-cards grid collapses to 1 column at tablet — use explicit `repeat(2, 1fr)` / `repeat(3, 1fr)` breakpoints.
- **T3** Glossary teal focus ring differs from the rest — fix after color tokens land.

### 8. Suggest-a-topic form
- **T2** No example placeholder for the description.
- **T2** No character counter / minimum length.
- **T3** Footer CTA blends with light theme backgrounds — add `border-left: 4px solid #0369a1`.
